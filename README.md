- React
- Axios
- Semantic UI
- CSS

# Build with:
- [YouTube API](https://developers.google.com/youtube/v3) 
- [Semantic UI](https://semantic-ui.com/) 

# Setup:
Clone this repo to your desktop and run `npm install` to install all the dependencies. Run the app by `npm start`
